# framehood (npm)

npm wrapper for the [Framehood CLI](https://github.com/Framehood/framehood-cli).
Installing this package downloads the prebuilt `framehood` binary for your
platform from the matching GitHub Release.

```sh
npm install -g framehood
framehood login
framehood generate "a red fox in the snow"
```

Full docs: <https://docs.framehood.ai>
